# **App Name**: Mentack Manufacturing

## Core Features:

- Multi-page Routing: Implementation of multi-page routing for Home, Products, Product Detail, Capabilities, Industries, About, Quality, and Contact pages.
- Bilingual Support (TR/EN): Implementation of bilingual support for Turkish and English languages, with a language toggle in the header and persisting language choice via localStorage.
- Translation Dictionary: All text content is managed through a translation dictionary for easy localization.
- SEO Optimization: Implementation of SEO-ready features for each page, including title, meta description, and OpenGraph tags.
- Image Optimization & Lazy Loading: Optimized images with lazy loading to improve page load times.
- Contact Form Validation: Contact form with input validation and professional success state upon submission.
- Product Detail Pages: Generation of product detail pages populated with product information using a slug.

## Style Guidelines:

- Background color: Light neutral (#F5F5F5) to maintain a clean and calm environment.
- Primary color: Dark blue/graphite (#34495E) to convey professionalism and reliability.
- Accent color: Muted steel/orange (#A0AEC0) used sparingly for subtle highlights and calls to action.
- Body and headline font: 'Inter', a sans-serif font (classification: sans-serif) is chosen for its corporate, modern look and high readability. Note: currently only Google Fonts are supported.
- Use a max-width container of approximately 1120-1200px to maintain visual consistency.
- Utilize simple line icons for a clean and modern aesthetic.
- Incorporate subtle animations for a polished user experience.
